Instructions:

This game serves to raise awareness on Web Content Accessibility Guidelines (WCAG) developed by the Web Accessibility Initiative (WAI) of the World Wide Web Consortium (W3C). The given program although syntactically correct would prove to be an accessiblity issue to some groups. Can you guess which criterion in accordance to WCAG does this program violate? Every wrongly guessed criteria will decrease Wi-Fi strength by 1 bar. Find out all accessibility issues in the given code before your Wi-Fi runs out!
